This page initiates an XHR before page load and injects the agent at the bottom of the page. Set the `script` query parameter to `async`, `defer`, `injection`, or `scriptTag` to control how the agent is loaded. Use the `loader` query parameter to control which agent is loaded and thereby which feature processes the ajax call. When using `injection`, the `injectionDelay` query parameter can be used to delay the injection of the agent by a set milliseconds.

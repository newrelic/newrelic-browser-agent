This page uses a button to initiate an XHR containing a function assigned to every possible event handler. Each event handler will attempt to synchronously wait for 50 milliseconds. After the XHR finishes, the hash will be updated so this page can be used for testing with and without the SPA feature. When running in a spec file, setting `window.disableAjaxHashChange = true` will disable the hash route change so the ajax call is fully processed by the ajax feature instead of the SPA feature. The page may appear to freeze when clicking the button!!!
